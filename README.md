# Bank Management System (C Project)

## 📌 Description

This is a simple Bank Management System developed in C.
It allows users to manage customer records such as adding, viewing, modifying, and deleting account details.

---

## ⚙️ Features

* Add new customer
* View customer details
* Modify customer information
* Delete customer record
* Data stored using file handling (`.dat` file)

---

## 📁 Project Files

* `main.c` → Main program
* `customer.c` → Customer-related functions
* `customer.h` → Header file
* `bms.exe` → Executable file
* `customers.dat` → Data file

---

## ▶️ How to Run

1. Download all files
2. Keep `bms.exe` and `customers.dat` in the same folder
3. Double-click `bms.exe` to run the program

---

## 🛠️ How to Compile (Optional)

If you want to compile manually:

```
gcc main.c customer.c -o bms
```

---

## 📦 Notes

* The `.dat` file is required for storing customer data
* Do not delete or move `customers.dat`, or the program may not work properly
